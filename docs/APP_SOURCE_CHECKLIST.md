# AppSource Submission Checklist

- [ ] `pbiviz package` produces a clean `.pbiviz`.
- [ ] Unique stable GUID.
- [ ] Versions match across `package.json`, `pbiviz.json`, CHANGELOG, release tag.
- [ ] `assets/icon.png` 300×300 transparent.
- [ ] Sample `.pbix` with usage metrics-style data.
- [ ] Privacy policy URL.
- [ ] Support URL.
- [ ] Terms of use URL.
- [ ] ≥3 screenshots (1280×720).
- [ ] Short + long description.
- [ ] No external network calls, `privileges` array empty.
- [ ] Accessibility tested: focus rings on tiles, aria-labels, high-contrast.
- [ ] Test cases: empty / override / full / missing fields / many failures / compact / resized.
