FABRIC ADOPTION MONITOR - INSTALL GUIDE
=======================================

This folder contains the ready-to-use Power BI custom visual file:
    fabricAdoptionMonitor*.1.0.0.0.pbiviz

You do NOT need Node.js, the pbiviz CLI, or any developer tooling to
install this visual. Just download the .pbiviz file and import it.


HOW TO INSTALL IN POWER BI DESKTOP
----------------------------------

1. Download the .pbiviz file from this folder.
   - In GitHub: click the filename, then click "Download raw file".

2. Open Power BI Desktop and open the report you want to add it to.

3. In the Visualizations pane on the right, click the three dots ("...")
   at the bottom of the icon grid.

4. Choose "Import a visual from a file".

5. Click OK on the safety warning, browse to the downloaded .pbiviz,
   and select it.

6. The Fabric Adoption Monitor icon will appear at the bottom of the
   Visualizations pane. Drag it onto your report canvas.

7. Bind your adoption measures (Active Users, Total Users, Report
   Views, Refresh Failures, Certified Datasets, etc.) or an Adoption
   Score measure if you compute the score elsewhere.


HOW TO INSTALL IN POWER BI SERVICE (powerbi.com)
------------------------------------------------

1. Open a report in Edit mode in the Power BI Service.

2. Visualizations pane -> three dots ("...") -> Import a visual from
   a file -> pick the downloaded .pbiviz.


TENANT POLICY BLOCKS THE IMPORT?
--------------------------------

Some Power BI tenants block uncertified custom visuals. If you see a
policy error during import, ask your Power BI administrator to add
this visual to the Organisational Visuals list:

    Power BI Admin Portal
        -> Tenant settings
        -> Organisational visuals
        -> Add a custom visual
        -> Upload the .pbiviz from this folder
